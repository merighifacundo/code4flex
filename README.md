

Code Generator for Flex
=======================

What is "Code Generator" ?
------------------------

"Code generator" is tool who gonna help you to generate code.


Compatibility
------------
PHP : 5.4, 5.5, 5.6, HHVM 3.3

Database driver : 

 - PostgreSQL
 - MySQL
 - Oracle
 - Flex


 
Installation
------------
Installation add it to the eclipse and be happy again
